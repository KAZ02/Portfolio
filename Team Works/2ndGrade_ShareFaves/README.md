### 概要
2022年度のチーム制作課題
コロナ禍において、旅行をしたいけど感染を危惧する人にむけた、観光地を提案するアプリケーションです。
※サーバーが開いていない可能性もございます。ご了承ください。

### 開発環境
- Visual Studio
- Visuatl Studio Code

### 使用言語
- C#
- HTML/CSS
- JavaScript

### 機能
- Webページ閲覧機能

### 操作方法
1. それぞれのエリアを選択し、都道府県を選択。
2. 遷移したWebページで紹介されている観光スポットの位置情報やHPのリンクをコピーすることができる。
