### 概要
2023年度の個人制作課題
経過日数を視覚的に分かりやすくしたカレンダーです。

### 使用言語
- C#

### 機能
- 一般的なカレンダー機能
- 経過日数表示機能
- イベント設定機能

### 操作方法
- カレンダーの操作方法
&emsp; 1. 「矢印」ボタンで表示する月を変更。
- イベントの設定方法
&emsp; 1. 「イベント設定」ボタンをクリックするとイベント設定画面が表示されます。
&emsp; 2. 「日時・項目・優先度」を入力したら「セットボタン」クリックし、ポップアップ表示の内容を確認してください。
&emsp; 3.トップページの「更新」ボタンを押すと、イベントの設定が完了します。
